# 01. LLM 모델 비교·선정 보고서

## 1. 비교 목적

본 프로젝트는 사회복지 강의 교안을 입력했을 때, 여러 LLM이 얼마나 정확하고 구조적으로 학습 노트를 생성하는지 비교하기 위해 수행하였다.

비교 과업은 **장애인복지론 1주차 1교시 「장애의 개념과 범주」 교안을 바탕으로 복습 및 시험 대비용 학습 노트를 생성하는 것**이다.

단순 요약이 아니라 다음 요소를 함께 평가하였다.

- 교안 내용 정확성
- 핵심 개념 추출 능력
- 강의 흐름 구조화
- 출력 형식 준수
- 시험 대비 활용성
- 현장 적용 설명력
- 환각 억제 능력
- 한국어 설명 명확성

---

## 2. 비교에 사용한 동일 입력

| 항목 | 내용 |
|---|---|
| 과목명 | 장애인복지론 |
| 주차/차시 | 1주차 1교시 |
| 강의주제 | 장애의 개념과 범주 |
| 자료 형태 | 강의 교안 |
| 주요 내용 | ICIDH, ICIDH-2, ICF, 오늘날의 장애 개념, OECD 장애 범주, 한국 장애 범주 확대 과정, 장애 인구 통계 |

세 모델에는 동일한 v2 프롬프트와 동일한 강의 교안을 입력하였다.

|---|---|---|---|---|

## 4. 평가 기준

각 항목은 1~5점으로 평가하였다.

| 평가 축 | 설명 |
|---|---|
| 교안 내용 정확성 | 교안의 개념, 연도, 수치, 용어를 정확히 반영했는가 |
| 핵심 개념 추출 능력 | ICIDH, ICIDH-2, ICF 등 핵심 개념을 잘 뽑았는가 |
| 강의 흐름 구조화 | 강의의 논리적 흐름을 보기 쉽게 정리했는가 |
| 출력 형식 준수 | 요청한 1~8번 출력 형식을 지켰는가 |
| 시험 대비 활용성 | 시험 포인트와 암기 사항을 잘 정리했는가 |
| 현장 적용 설명력 | 사회복지 현장 또는 정신건강사회복지 관점으로 연결했는가 |
| 환각 억제 | 교안에 없는 내용을 사실처럼 단정하지 않았는가 |
| 한국어 설명 명확성 | 학습자가 읽기 쉽게 명확한 문장으로 정리했는가 |

## 5. 모델별 평가

### 5-1. Gemini 3 Flash

| 평가 축 | 점수 | 근거 |
|---|---:|---|
| 교안 내용 정확성 | 4 | ICIDH, ICIDH-2, ICF, 장애 범주 확대 과정은 대체로 정확히 반영하였다. |
| 핵심 개념 추출 능력 | 4 | 장애 개념 변화, 사회적 모델, OECD 범주, 한국 범주 확대를 잘 추출하였다. |
| 강의 흐름 구조화 | 4 | 전체 흐름은 잡았으나 세부 목차 반영은 상대적으로 부족하였다. |
| 출력 형식 준수 | 5 | v2 프롬프트에서 요구한 1~8번 형식을 모두 따랐다. |
| 시험 대비 활용성 | 4 | 시험 포인트를 제시했으나 세부 암기 항목은 다소 간략하였다. |
| 현장 적용 설명력 | 4 | 정신건강사회복지 관점과 사회적 낙인을 연결하였다. |
| 환각 억제 | 3 | 한국 장애인 출현율이 낮은 이유를 장애 범주의 엄격성 때문이라고 단정하는 등 교안 밖 해석이 일부 포함되었다. |
| 한국어 설명 명확성 | 5 | 문장이 자연스럽고 초보 학습자가 이해하기 쉬웠다. |

#### 총평

Gemini 3 Flash는 간결하고 읽기 쉬운 학습 노트를 생성하였다. 주요 개념을 빠르게 정리하는 데는 적합했지만, 교안의 세부 내용 일부가 생략되었고 교안에 명시되지 않은 원인 해석을 단정하는 문제가 있었다.

---

### 5-2. Claude Haiku 4.5

| 평가 축 | 점수 | 근거 |
|---|---:|---|
| 교안 내용 정확성 | 4 | 주요 개념과 연도, 범주 확대 과정은 대체로 정확하게 반영하였다. |
| 핵심 개념 추출 능력 | 5 | ICIDH, ICIDH-2, ICF, 사회적 모델, 한국 장애 범주를 충실히 추출하였다. |
| 강의 흐름 구조화 | 5 | 흐름도와 목차 구성이 명확했고 학습노트로 보기 좋았다. |
| 출력 형식 준수 | 5 | 요청한 1~8번 항목을 모두 완성하였다. |
| 시험 대비 활용성 | 5 | 시험 포인트, 암기 사항, 헷갈리는 개념 비교가 잘 정리되었다. |
| 현장 적용 설명력 | 4 | 정신건강사회복지 관점까지 연결했으나 일부는 교안 밖 해석이었다. |
| 환각 억제 | 3 | 한국의 장애인 출현율이 낮은 이유를 장애 범주가 좁기 때문이라고 단정하는 등 교안 밖 해석이 일부 포함되었다. |
| 한국어 설명 명확성 | 5 | 문장, 표, 요약 구조가 명확하고 학습용으로 적합하였다. |

#### 총평

Claude Haiku 4.5는 학습노트 완성도와 구조화 능력이 가장 균형적이었다. ICIDH, ICIDH-2, ICF의 차이를 표로 정리하고 시험 포인트와 헷갈리기 쉬운 부분까지 안정적으로 제시하였다. 일부 교안 밖 해석은 있었지만, 전체 출력이 완결되어 실제 복습 자료로 활용하기 가장 적합했다.

---

### 5-3. GPT-5.5

| 평가 축 | 점수 | 근거 |
|---|---:|---|
| 교안 내용 정확성 | 5 | 교안의 개념, 연도, 수치, 용어를 매우 정확하게 반영하였다. |
| 핵심 개념 추출 능력 | 5 | 손상, 능력장애, 사회적 불리, 활동, 참여, 상황 요인 등을 세밀하게 추출하였다. |
| 강의 흐름 구조화 | 5 | 강의 정보 확인, 흐름도, 세부 개념 정리가 매우 체계적이었다. |
| 출력 형식 준수 | 2 | 1~8번 출력 형식을 끝까지 완성하지 못하고 중간에 끊겼다. |
| 시험 대비 활용성 | 4 | 시험 포인트 자체는 좋았으나 응답이 중간에 끊겨 후반부가 누락되었다. |
| 현장 적용 설명력 | 1 | 현장 적용 항목까지 충분히 도달하지 못했다. |
| 환각 억제 | 5 | OECD 출현율 차이의 원인을 단정하지 않고 교안에 없는 내용은 확인 필요로 처리하였다. |
| 한국어 설명 명확성 | 5 | 문장과 표 정리는 매우 명확하였다. |

#### 총평

GPT-5.5는 세 모델 중 교안 근거와 외부 해석을 가장 엄밀하게 구분하였다. 특히 OECD 장애인 출현율에서 한국과 일본의 수치가 낮게 제시된 이유를 임의로 단정하지 않고, 교안에 제시된 사실까지만 정리한 점은 환각 억제 측면에서 우수했다.

그러나 동일한 v2 프롬프트를 입력했을 때 출력이 중간에 끊겼고, 현장 적용, 헷갈리기 쉬운 부분, 확인 필요 사항, 최종 정리까지 완성하지 못했다. 따라서 정확성과 근거 구분은 우수하지만 긴 학습노트 자동 생성 업무에서는 출력 완결성과 안정성에 한계가 있었다.

## 7. 최종 선정 모델

최종 모델은 **Claude Haiku 4.5**로 선정하였다.

### 선정 이유

1. **출력 완결성이 가장 안정적이었다.**  
   GPT-5.5는 긴 답변 중간에 끊기는 문제가 있었다. 이번 업무 과업은 “완성된 학습노트 자동 생성”이므로 출력 완결성이 중요하다.

2. **학습노트 구조화와 시험 대비성이 가장 균형적이었다.**  
   Claude Haiku 4.5는 ICIDH, ICIDH-2, ICF의 개념 비교, 시험 포인트, 헷갈리는 부분, 최종 정리를 모두 포함하였다.

3. **실제 업무 적용성이 높았다.**  
   Gemini는 간결하지만 세부성이 부족했고, GPT-5.5는 정확하지만 출력이 중단되었다. 반면 Claude Haiku 4.5는 구조화, 가독성, 완결성, 시험 대비 활용성 측면에서 가장 균형 잡힌 결과를 제공하였다.

## 8. 결론

세 모델 모두 사회복지 강의 교안을 학습노트로 정리하는 데 일정 수준 이상의 성능을 보였다.

- Gemini 3 Flash는 간결성과 속도 면에서 유리했다.
- GPT-5.5는 정확성과 환각 억제 측면에서 가장 우수했다.
- Claude Haiku 4.5는 출력 완결성과 학습노트 활용성에서 가장 균형적이었다.

따라서 본 프로젝트에서는 **Claude Haiku 4.5**를 사회복지 강의 교안 기반 학습노트 자동 생성 업무에 가장 적합한 모델로 선정하였다.
