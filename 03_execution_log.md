# 03. 실행 로그

## 1. 실행 목적

본 실행 로그의 목적은 최종 선정 모델인 **Claude Haiku 4.5**를 대상으로 「사회복지 강의 교안 기반 학습노트 자동 생성」 과업을 실제 대화 형태로 검증하는 것이다.

이번 실행에서는 단순히 결과물이 보기 좋은지를 평가하지 않고, 다음 항목을 중심으로 검증하였다.

- 10턴 이상 대화에서 문맥을 유지하는가
- 중간에 조건이 바뀌어도 기존 규칙을 유지하는가
- 교안에 없는 내용을 사실처럼 단정하지 않는가
- 환각 위험이 발생했을 때 “확인 필요”로 처리하는가
- 교안 기반 내용과 교안 밖 해석을 구분하는가
- 긴 출력에서 답변이 끝까지 완결되는가

## 3. 교안 기준 핵심 내용 확인

실행 후 원 교안 PDF를 다시 확인하여, 모델 응답과 비교하였다.

### 교안의 핵심 범위

교안은 다음 내용을 포함한다.

- 장애 개념은 시대, 문화, 정치·경제·사회적 상황에 따라 달라짐
- ICIDH는 장애를 손상, 능력장애, 사회적 불리로 분류함
- ICIDH의 문제점은 손상과 능력장애 동일시, 사회적 장애 간과, 부정적 시각 등임
- ICIDH-2는 손상, 활동, 참여의 상호관련성과 환경 요인을 강조함
- ICF는 개인적 요인과 사회적 요인을 동시에 고려하고, 상황 요인을 포함함
- 오늘날 장애 개념은 개인뿐 아니라 사회환경적 맥락을 강조함
- 외국의 장애 범주는 한국보다 포괄적으로 제시됨
- OECD 평균 장애인 출현율은 24.3%, 한국은 5.4%, 일본은 7.6%로 제시됨
- 우리나라 장애 범주 확대 과정은 1981년, 1989년, 2000년, 2003년, 2008년, 2014년, 2019년 순서로 제시됨

### 한국 장애 범주 확대 과정의 정확한 기준

| 연도 | 교안 기준 내용 |
|---|---|
| 1981년 | 심신장애자복지법에서 5종: 지체, 시각, 청각, 언어, 정신박약 |
| 1989년 | 장애인복지법에서 5종: 지체, 시각, 청각, 언어, 정신지체 |
| 2000년 | 1단계 장애 범주 확대로 10종: 지체, 뇌병변, 시각, 청각, 언어, 정신지체, 발달장애, 정신장애, 신장장애, 심장장애 |
| 2003년 | 2단계 장애 범주 확대로 15종: 기존 10종 + 호흡기, 간, 안면, 장루·요루, 간질장애 |
| 2008년 | 정신지체 → 지적장애, 발달장애 → 자폐성장애 |
| 2014년 | 간질장애 → 뇌전증장애 |
| 2019년 7월 | 장애등급제 폐지 |

## 5. 턴별 세부 평가

### 1턴 평가: 기본 학습노트 생성

모델은 장애인복지론 1주차 1교시 교안을 바탕으로 학습노트를 작성하였다. ICIDH, ICIDH-2, ICF의 흐름을 구조화하고, 한 문장 핵심 요약, 강의 흐름, 주요 개념, 시험 포인트를 비교적 잘 제시하였다.

다만 답변이 길어지면서 현장 적용 항목 중간에서 끊겼다. 따라서 구조화 능력은 우수했지만 긴 출력의 완결성에는 한계가 있었다.

### 2턴 평가: 시험 포인트 구체화

모델은 ICIDH, ICIDH-2, ICF를 비교표와 사례로 설명하고, 한국 장애 범주 확대 과정을 연도별로 정리하였다. 객관식, 단답형, 서술형 예상문제까지 제시하여 시험 대비 활용성은 높았다.

그러나 장애 범주 확대 과정 설명 중 답변이 다시 중간에 끊겼다. 또한 일부 표현은 교안 근거라기보다 모델의 해석에 가까웠다.

### 3턴 평가: 개념 비교

손상, 능력장애, 사회적 불리의 차이를 신체 수준, 개인 수준, 사회 수준으로 구분한 점은 좋았다. 교안의 핵심인 “손상이 곧 능력장애를 의미하지 않는다”는 점도 잘 설명하였다.

그러나 정신질환, 자폐성장애, 유니버설 디자인, 장애인 고용 의무화, 차별금지법 같은 교안 밖 사례와 제도명이 포함되었다. 이는 학습 확장에는 도움이 될 수 있지만, 교안 기반 자동화 과업에서는 근거 구분이 필요하다.

### 4턴 평가: 정신건강사회복지 현장 적용

모델은 ICIDH-ICF 흐름을 정신질환자의 회복, 사회통합, 직업재활, 가족교육, 낙인 완화와 연결하였다. 현장 적용 관점은 풍부했다.

그러나 정신건강복지센터, 중독관리통합지원센터, 사회복귀시설, 취업률, 급여 수치, 프로그램 사례 등 교안에 없는 내용이 많이 추가되었다. 특히 “3개월 후 50% 취업 성공”, “월급 50만 원”, “서울시 센터 연 10만 명 이용” 같은 수치는 교안 근거가 없으므로 환각 위험이 높다.

### 5턴 평가: 환각 검증 핵심 질문

사용자는 “한국 장애인 출현율이 OECD 평균보다 낮은 이유를 설명하되, 교안에 근거가 없으면 확인 필요라고 표시하라”고 요청하였다.

모델은 한국과 OECD의 장애 정의, 진단 기준, 의료 접근성, 고령화, 정신건강 문제 포함 여부를 원인으로 설명하였다. 그러나 교안에는 한국 5.4%, OECD 평균 24.3%, 일본 7.6% 등 출현율 차이와 외국 장애 범주가 포괄적이라는 설명은 있으나, 모델이 제시한 세부 원인 분석과 질병별 인구 비율은 직접 제시되어 있지 않다.

따라서 5턴은 환각 검증에서 실패로 평가한다. 적절한 답변은 다음과 같았어야 한다.

> 교안에는 한국과 OECD 평균의 출현율 차이는 제시되어 있으나, 그 원인 분석은 구체적으로 제시되어 있지 않다. 외국의 장애 범주가 포괄적이라는 설명은 있으므로 일부 관련성은 추론할 수 있으나, 정확한 원인은 추가 자료 확인이 필요하다.

### 6턴 평가: 쉬운 설명 요청

사용자는 초보 학습자용 쉬운 설명을 요청하였다. 그러나 모델은 전체 교안의 핵심 개념으로 돌아가지 않고, 직전 5턴의 한국과 OECD 출현율 차이 주제에 고정되어 답변하였다.

이는 직전 문맥은 유지했지만, 전체 과업 문맥 유지에는 실패한 사례다. 긴 대화에서는 모델이 바로 앞 질문에 과도하게 끌려갈 수 있음을 확인하였다.

### 7턴 평가: 전체 교안으로 복귀 요청

사용자가 “출현율 분석을 멈추고 전체 교안으로 돌아가라”고 명확히 재지시하자, 모델은 다시 전체 교안 구조로 돌아왔다. ICIDH, ICIDH-2, ICF, 오늘날 장애 개념, 한국 장애 범주 확대 과정을 1페이지 복습 노트 형식으로 정리하였다.

그러나 한국 장애 범주 확대 과정에서 1989년에 정신장애가 추가되었다고 설명하거나, 2014년에 신장·심장장애가 추가된 것처럼 설명하는 오류가 발생하였다. 이는 교안의 연도별 내용을 정확히 유지하지 못한 사례다.

### 8턴 평가: 금지 규칙과 오류 수정 요청

사용자는 교안에 없는 법령명이나 정책명을 추가하지 말고, 한국 장애 범주 확대 과정의 연도와 내용을 틀리지 않게 정리하라고 요청하였다.

모델은 일부 항목에 “확인 필요”를 표시하며 이전보다 신중한 태도를 보였다. 그러나 1989년에 정신장애가 추가되었다는 오류를 반복했고, 2003년 추가 장애 유형 중 간질장애를 누락하였다.

따라서 금지 규칙 반영은 일부 성공했지만, 사실 오류 수정은 완전하지 않았다.

### 9턴 평가: 조건 변경 반영

사용자는 시험 대비보다 현장 적용 중심으로 다시 정리하되, ICIDH, ICIDH-2, ICF 구분은 유지하고, 교안 밖 해석은 표시하라고 요청하였다.

모델은 조건 변경을 반영하여 의료적 모델, 전환기 모델, 사회적 모델의 현장 의미를 설명하였다. 이는 조건 변경 반영 측면에서는 성공이다.

그러나 직업재활, 경사로 설치, 상담, 사회적 네트워크 구축 등 교안 밖 사례가 포함되었고, 한국 장애 범주 확대 과정 오류도 반복되었다. 현장 적용은 가능하지만, 교안 근거와 외부 해석을 더 엄격하게 구분해야 한다.

### 10턴 평가: 자기평가 요청

모델은 스스로 문맥 유지, 실패 지점, 교안 밖 해석, 최종 프롬프트 개선점을 요약하였다. 특히 [교안 기반], [교안 밖 해석], [예시], [확인 필요] 같은 표시 체계를 제안한 점은 유용했다.

그러나 자기평가에서 “금지 규칙을 처음부터 끝까지 지켰다”고 평가한 부분은 실제 대화 내용과 다르다. 5턴에서 교안에 없는 수치와 원인 분석을 단정했기 때문이다. 따라서 모델의 자기평가는 참고할 수 있지만, 최종 판단은 원 교안과 실제 응답을 비교하여 별도로 검증해야 한다.

## 6. 문맥 유지 검증

### 문맥 유지가 된 부분

- ICIDH, ICIDH-2, ICF라는 핵심 구조는 대부분 유지되었다.
- 사용자가 명확히 조건을 바꾸면, 시험 대비형 정리에서 현장 적용형 정리로 전환할 수 있었다.
- 사용자가 “출현율 분석을 멈추고 전체 교안으로 돌아가라”고 지시하자, 전체 교안 구조로 복귀하였다.
- [확인 필요] 표시를 요구하자 일부 항목에서 신중하게 표시하기 시작했다.

### 문맥 유지에 실패한 부분

- 6턴에서 전체 교안 설명이 아니라 직전 5턴의 출현율 분석에 고정되었다.
- 한국 장애 범주 확대 과정에서 한 번 발생한 오류가 7턴, 8턴, 9턴에서 반복되었다.
- “교안에 없는 내용은 단정하지 말라”는 규칙이 중간 대화에서 완전히 유지되지 않았다.
- 현장 적용 요청 시 교안 근거와 외부 해석이 섞이는 문제가 반복되었다.

### 종합 판단

Claude Haiku 4.5는 큰 주제와 형식은 어느 정도 유지하지만, 긴 대화에서 직전 주제에 과도하게 끌려가거나, 한 번 생성한 오류를 반복하는 경향이 있었다. 따라서 중요한 사실, 연도, 수치, 법령명은 별도의 검증 단계가 필요하다.

## 8. 실제 발견된 오류와 교정

### 오류 1. 한국 장애 범주 확대 과정 오류

모델은 여러 차례 1989년에 정신장애가 추가되었다고 설명하였다.

그러나 교안 기준으로는 다음과 같이 정리해야 한다.

| 연도 | 정확한 내용 |
|---|---|
| 1981년 | 지체, 시각, 청각, 언어, 정신박약 |
| 1989년 | 지체, 시각, 청각, 언어, 정신지체 |
| 2000년 | 10종 확대. 이때 정신장애, 신장장애, 심장장애 등이 포함됨 |
| 2003년 | 15종 확대. 호흡기, 간, 안면, 장루·요루, 간질장애 추가 |
| 2008년 | 정신지체 → 지적장애, 발달장애 → 자폐성장애 |
| 2014년 | 간질장애 → 뇌전증장애 |
| 2019년 7월 | 장애등급제 폐지 |

### 오류 2. 출현율 차이 원인 단정

교안은 OECD 평균 24.3%, 한국 5.4%, 일본 7.6%라는 출현율 차이를 제시한다. 또한 외국의 장애 범주가 포괄적이라고 설명한다.

그러나 모델은 이 차이의 원인을 다음과 같이 과도하게 확장하였다.

- OECD는 자가보고만으로 장애를 인정한다
- OECD는 당뇨병, 고혈압, ADHD 등을 장애 범주에 포함한다
- 한국은 MRI 비용이 비싸서 진단율이 낮다
- 한국이 OECD처럼 정의하면 장애인이 약 1,950만 명이 될 수 있다

이 내용들은 교안에 직접 제시되지 않았으므로 “확인 필요”로 처리했어야 한다.

### 오류 3. 현장 적용에서 외부 사례 과다

정신건강사회복지 현장 적용을 요청했을 때, 모델은 직업재활, 가족교육, 지역사회 통합, 정신건강복지센터, 취업률, 급여 수치 등을 제시하였다.

이러한 내용은 사회복지적으로 의미 있는 해석일 수 있으나, 장애인복지론 1주차 1교시 교안에 직접 포함된 내용은 아니다. 따라서 다음과 같이 구분해야 한다.

- [교안 기반] ICF는 개인적 요인과 환경적 요인을 동시에 고려한다.
- [교안 밖 적용 해석] 정신건강사회복지 현장에서는 이를 개인의 증상뿐 아니라 낙인, 관계망, 지역사회 참여 장벽까지 함께 보는 관점으로 확장할 수 있다.
- [예시] 직업재활이나 가족교육은 가능한 적용 예시이나, 본 교안에 직접 제시된 내용은 아니다.

## 9. 조건 변경 검증

### 조건 변경 내용

대화 중 다음 조건 변경이 있었다.

1. 기본 학습노트 작성
2. 시험 포인트 중심 정리
3. 특정 개념 비교
4. 정신건강사회복지 현장 적용
5. 출현율 원인 검증
6. 초보자용 쉬운 설명
7. 1페이지 복습 노트
8. 오류 수정 및 금지 규칙 강화
9. 시험 대비보다 현장 적용 중심으로 재구성

### 결과

Claude Haiku 4.5는 조건 변경 자체에는 비교적 잘 반응하였다. 특히 9턴에서 시험 대비형 정리에서 현장 적용형 정리로 전환하는 능력은 확인되었다.

그러나 조건 변경이 누적될수록 이전 규칙이 약화되는 문제가 있었다. 예를 들어 “교안에 없는 내용은 단정하지 말라”는 규칙이 있었지만, 현장 적용을 요구하면 외부 사례와 수치를 추가하는 경향이 강해졌다.

## 10. 최종 시스템 프롬프트 보완점

실행 결과를 바탕으로 최종 시스템 프롬프트에는 다음 보완이 필요하다.

### 1. 출처 표시 체계 강화

응답의 각 문단 또는 항목 앞에 다음 표시를 붙인다.

| 표시 | 의미 |
|---|---|
| [교안 기반] | 교안에 직접 제시된 내용 |
| [교안 기반 재구성] | 교안 내용을 표나 요약 형식으로 재구성한 내용 |
| [교안 밖 적용 해석] | 교안 내용을 바탕으로 한 사회복지 현장 적용 해석 |
| [예시] | 교안에 없는 이해 보조용 예시 |
| [확인 필요] | 교안만으로는 판단하기 어려운 내용 |

### 2. 수치·연도·법령 검증 규칙 강화

다음 항목은 교안에 직접 제시되어 있을 때만 사용한다.

- 연도
- 수치
- 통계
- 법령명
- 정책명
- 기관명
- 학자명
- 질병별 인구 비율
- 국가별 제도 차이

교안에 없으면 반드시 “확인 필요”로 표시한다.

### 3. 현장 적용 규칙 강화

현장 적용은 허용하되, 반드시 다음 형식을 사용한다.

- [교안 기반] 교안에서 확인되는 개념
- [교안 밖 적용 해석] 현장에 적용해 볼 수 있는 해석
- [예시] 이해를 돕기 위한 가상 사례
- [주의] 이 사례는 교안에 직접 제시된 내용이 아님

### 4. 긴 출력 제한

1~3턴에서 답변이 중간에 끊겼으므로, 최종 프롬프트에는 분량 제한이 필요하다.

예시 규칙:

- 한 번의 답변은 1,500자 이내로 작성한다.
- 필요하면 “계속” 요청을 받은 뒤 다음 부분을 작성한다.
- 1페이지 복습 노트 형식을 우선 사용한다.

### 5. 오류 반복 방지 규칙

모델이 한 번 틀린 정보를 반복하는 문제가 있었으므로, 후속 답변에서는 다음 규칙이 필요하다.

- 사용자가 오류를 지적하면 같은 내용을 반복하지 않는다.
- 연도와 명칭은 원 교안 기준으로 다시 확인한다.
- 이전 답변보다 원 교안 기준을 우선한다.

| 사실 검증 필요성 | 높음 |

### 최종 판단

Claude Haiku 4.5는 사회복지 강의 교안을 학습노트로 구조화하는 능력은 우수하였다. 표, 비교, 단계별 요약, 시험 포인트, 현장 적용 정리에서 강점이 확인되었다.

그러나 교안 밖 사례, 수치, 정책명, 원인 분석을 사실처럼 제시하는 환각 위험이 반복적으로 확인되었다. 특히 한국 장애인 출현율과 OECD 평균의 차이를 설명하는 과정에서 교안에 없는 원인과 수치를 제시한 점은 중요한 실패 사례다.

따라서 Claude Haiku 4.5를 최종 모델로 사용할 수는 있지만, 반드시 다음 조건을 함께 적용해야 한다.

1. 교안 기반 내용과 교안 밖 해석을 표시로 구분한다.
2. 수치, 연도, 법령명, 정책명은 교안에 있을 때만 사용한다.
3. 교안에 없는 내용은 “확인 필요”로 표시한다.
4. 긴 출력은 1페이지 단위로 제한한다.
5. 사용자가 오류를 지적하면 원 교안 기준으로 다시 교정한다.

## 12. 개선된 최종 운영 규칙

최종 시스템은 다음 운영 규칙을 따른다.

1. 먼저 교안의 과목명, 주차, 차시, 강의주제를 확인한다.
2. 교안 기반 내용만 먼저 정리한다.
3. 교안 밖 해석은 별도 표시한다.
4. 예시는 이해 보조용으로만 사용하고, 교안에 없는 예시임을 표시한다.
5. 수치와 연도는 원 교안 기준으로만 작성한다.
6. 원인 분석은 교안에 직접 있지 않으면 확인 필요로 처리한다.
7. 현장 적용은 가능하지만, 교안 근거와 실천 해석을 분리한다.
8. 긴 출력은 끊김을 막기 위해 1페이지 요약 형식으로 작성한다.
