# 03. 실행 로그

## 1. 실행 목적

본 실행 로그의 목적은 최종 선정 모델인 **Claude Haiku 4.5**를 대상으로 사회복지 강의 교안 기반 학습노트 생성 과업을 실제 대화 형태로 검증하는 것이다.

검증 초점은 다음과 같다.

- 10턴 이상 대화에서 문맥을 유지하는가
- 중간에 조건이 바뀌어도 기존 요구사항을 반영하는가
- 교안에 없는 내용을 임의로 만들지 않는가
- 출력 형식과 금지 규칙을 유지하는가
- 환각 위험이 발생했을 때 “확인 필요”로 처리하는가

## 4. 문맥 유지 검증

### Pass 기준

문맥 유지가 잘 된 것으로 판단하려면 다음 기준을 만족해야 한다.

- 이전에 정한 출력 형식이 대화 후반까지 유지되어야 한다.
- “교안에 없는 내용은 단정하지 않기”라는 금지 규칙이 유지되어야 한다.
- 중간에 조건이 바뀌면 바뀐 조건을 반영하되, 기존 핵심 개념은 유지해야 한다.
- ICIDH, ICIDH-2, ICF의 개념 구분이 대화 중 계속 일관되어야 한다.

### 검증 결과

Claude Haiku 4.5는 10턴 대화에서 전체적으로 문맥을 유지하였다. 특히 장애 개념 변화, ICIDH와 ICIDH-2의 차이, ICF의 상황 요인, 한국 장애 범주 확대 과정은 반복 요청에서도 비교적 일관되게 유지되었다.

다만 현장 적용을 요구했을 때, 교안에 직접 제시되지 않은 “강점 관점”이나 “성별 맞춤형 접근” 같은 외부 실천 개념이 포함될 가능성이 있었다. 따라서 시스템 프롬프트에서 “교안 근거”와 “교안 밖 현장 적용 해석”을 더 분명히 구분하도록 보완할 필요가 있다.

## 5. 조건 변경 검증

### 조건 변경 상황

대화 중간에 다음과 같은 조건 변경을 적용하였다.

1. 시험 대비 중심 정리
2. 초보 학습자용 쉬운 설명
3. 1페이지 복습 노트 형식
4. 현장 적용 중심 정리
5. 교안 밖 법령명, 정책명 추가 금지

### 결과

Claude Haiku 4.5는 조건 변경을 대체로 잘 반영하였다. 특히 시험 대비 중심 요청에서는 연도, 개념 비교, 암기 포인트를 중심으로 정리했고, 초보 학습자용 요청에서는 전문용어를 비교적 쉬운 표현으로 풀어 설명하였다.

그러나 현장 적용 중심으로 바뀌었을 때 교안 밖 실천 개념이 들어갈 가능성이 있어, 최종 시스템 프롬프트에는 다음 규칙이 필요하다.

> 현장 적용은 가능하지만, 교안에 직접 나온 내용과 외부 해석을 반드시 구분한다.

## 7. 문제 발생 지점

실행 과정에서 확인한 주요 문제는 다음과 같다.

### 1. 교안 밖 해석 단정 가능성

일부 모델은 한국 장애인 출현율이 OECD 평균보다 낮은 이유를 “장애 범주가 좁기 때문”이라고 단정하였다. 그러나 교안에는 출현율 수치만 제시되어 있고, 구체적 원인 분석은 제시되어 있지 않다.

따라서 이 부분은 다음과 같이 수정해야 한다.

> 한국의 장애인 출현율은 OECD 평균보다 낮게 제시되어 있다. 다만 그 원인이 장애 범주의 엄격성 때문인지는 교안 내용만으로는 확인이 필요하다.

### 2. 현장 적용에서 외부 개념 포함

정신건강사회복지 관점으로 확장할 때 “강점 관점”, “성별 맞춤형 접근” 같은 개념이 포함될 수 있다. 이는 사회복지 실천상 의미 있는 해석이지만, 교안에 직접 나온 내용은 아니다.

따라서 다음과 같이 구분해야 한다.

> 교안 근거: 사회적 낙인과 따돌림은 사회참여 제한이 될 수 있다.  
> 교안 밖 현장 적용 해석: 이는 정신장애인의 지역사회 통합과 낙인 완화 실천으로 연결해 볼 수 있다.

### 3. 긴 출력의 안정성 문제

GPT-5.5는 정확성과 환각 억제 측면에서 우수했지만, 긴 학습노트 생성 중 출력이 중간에 끊기는 문제가 있었다. 따라서 긴 결과물을 요구할 때는 분량 제한을 두거나 “1페이지 요약”처럼 출력 범위를 좁히는 전략이 필요하다.

## 8. 수정 및 개선 결과

실행 결과를 바탕으로 최종 프롬프트에는 다음 규칙을 강화하였다.

- 교안에 없는 원인 분석은 “확인 필요”로 표시한다.
- 현장 적용은 가능하지만 교안 근거와 교안 밖 해석을 구분한다.
- 시험 포인트는 “반드시 출제”라고 단정하지 않는다.
- 법령명, 정책명, 학자명, 수치는 교안에 없으면 임의로 추가하지 않는다.
- 긴 출력이 필요한 경우 표와 bullet을 활용해 압축적으로 정리한다.

## 9. 최종 판단

10턴 이상 대화 검증 결과, Claude Haiku 4.5는 강의 교안 기반 학습노트 생성 과업에서 출력 완결성, 구조화, 시험 대비 활용성 측면에서 가장 안정적인 결과를 보였다.

다만 환각 위험을 줄이기 위해서는 다음 보완이 필요하다.

1. 교안 근거와 외부 해석 구분
2. 확인 필요 표시 강화
3. 긴 출력 시 분량 제한
4. 현장 적용 시 추론 범위 명시

따라서 최종 시스템은 Claude Haiku 4.5를 기본 모델로 사용하되, 환각 검증 규칙을 강화한 v2 프롬프트를 적용하는 방식이 적절하다고 판단하였다.
